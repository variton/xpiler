# Xpiler - X compiler
A wrapper around main stream compilers.
It's main purpose is to facilitate the compiling of software components such as:

* cxx
* rust
* python
* c 

*remark*:
    *the tools should be developed and tested on ubuntu 22.02,24.04 only*

## Install
Pull the repository and add your repository to your path:

```
git clone https://github.com/variton/xpiler.git .

```
*remark*:
    *add to the linux PATH the path to the tool you want to use*

## Vim plugin

This vim plugin that allows to compile:

* small projects
* snippets
* projects
* software components
